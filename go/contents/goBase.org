* Go 基础
** 不需要添加分号
不需要在语句或声明的末尾添加分号, 除非一行上有多条语句. 编译器会主动把特定符号后的换行符转换为
分号, 因此换行符添加的位置会影响Go代码的正确解析.

例如: x + y, 可在+后换行, 不能在"+"前换行.

** ++, --
go中++和--是语句, 不是表达式, 因此不能用于赋值.

** for
for init; condition; post {
}

for的这三个部分都可以省略或省略部分. condition是一个布尔表达式, 其值在每次循环迭代开始时计算.
